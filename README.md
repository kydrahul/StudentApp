# IIITNR Attendance - Student App

The mobile application for students to view their schedule, mark attendance via QR code, and track their attendance history.

## Features
- **Dashboard**: View today's classes and overall attendance percentage.
- **QR Attendance**: Scan dynamic QR codes generated by faculty to mark attendance.
- **Geofencing**: Location-based validation ensures students are in class/campus.
- **Timetable**: Weekly schedule view.
- **Profile**: Manage student profile details.

## Tech Stack
- **Framework**: Flutter (Dart)
- **State Management**: standard setState/FutureBuilder
- **Backend**: Node.js, Firebase
- **Maps**: Geolocator, Google Maps (if applicable)

## Setup & Build
1.  Ensure you have Flutter installed (`flutter doctor`).
2.  Clone the repository:
    ```bash
    git clone https://github.com/kydrahul/StudentApp.git
    cd StudentApp
    ```
3.  Install dependencies:
    ```bash
    flutter pub get
    ```
4.  Run on device/emulator:
    ```bash
    flutter run
    ```
5.  Build APK:
    ```bash
    flutter build apk --release
    ```
